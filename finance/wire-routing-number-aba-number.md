# Wire Routing Number Is Called Many Things

For U.S. domestic wire transfers, a _Wire Routing Number_ is needed to setup a wire recipient.
This number is the same as the _ABA #_.
Wire Routing number is sometimes called the _Bank Routing Number_ if that is listed under the domestic Wire payments section.
The Wire Routing number is usually different from the _ACH Routing Number_.
The number should be nine (9) digits.
