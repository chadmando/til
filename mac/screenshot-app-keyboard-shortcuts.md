# Screenshot App Keyboard Shortcuts

|Action | Shortcut |
| :-- | :-: |
| Capture the Entire Screen | Shift+Cmd+3 |
| Capture a Portion of the Screen | Shift+Cmd+4 |
| Capture a Window or the Menu Bar | Shift+Cmd+5 |
| Open Screenshot App | Shift+Cmd+6 |
