# Manually Boot To Audit Mode

At the Out Of Box Experience (OOBE) press: `Ctrl + Shift + F3` [^1]

## Dell Laptops

For some Dell laptops, you may need to include the `fn` key.
Making the combination `Ctrl + Shift + fn + F3`
